# CLI Reference

- [diffulex.server](diffulex.server.md)
- [diffulex.bench](diffulex.bench.md)

These pages summarize the command-line entry points exposed by the repository.
The server CLI is for online HTTP serving. The benchmark CLI is for repeatable
evaluation through lm-evaluation-harness.

Use `--help` for the exact parser output in the current checkout. The reference
pages group the most important flags by purpose and explain how they interact
with Diffulex configuration.
